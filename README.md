# online-m-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放一些独立的、功能相对完整的 HTML 页面。这些页面通常以单文件形式存在，不依赖外部复杂框架或服务端环境，可直接在浏览器中打开运行。

页面内容涵盖各类实用小工具、演示、实验性交互、信息展示等。每个页面均以独立目录或文件形式组织，方便单独查看和引用。

## 目录说明

```
├── pages/          # 页面存放目录，每个子文件夹或文件对应一个独立页面
├── assets/         # 公共资源文件（如样式表、脚本、图片等），供多个页面复用
├── index.html      # 仓库页面导航索引（可选）
└── README.md       # 本文件
```

- `pages/`：主要页面存放位置。每个页面可能包含自己的 HTML、CSS、JavaScript 文件，或直接以单 HTML 文件形式存在。
- `assets/`：存放多个页面可能共用的资源，避免重复。具体资源文件见该目录下的说明。
- `index.html`：提供一个简单的页面导航，方便浏览仓库中所有可用的 HTML 页面。

## 页面归档说明

本仓库中的页面主要来自以下来源：

- 个人开发或实验的 HTML 页面
- 部分公开可用的单页面工具或演示
- 其他独立、无依赖的 HTML 资源

每个页面在归档时会尽量保持原始功能与结构，必要时会添加注释说明来源或修改记录。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 目录下的 HTML 文件，或通过 `index.html` 导航访问。
3. 部分页面可能需要本地服务器环境（如涉及 AJAX 请求或模块加载），建议使用简单的 HTTP 服务器（如 `python -m http.server`）进行预览。

## 维护说明

- 本仓库不定期更新，新增页面会按类型或功能放入对应目录。
- 欢迎提交 Issue 或 Pull Request，建议或贡献新的独立 HTML 页面。
- 提交的页面应尽量保持独立、无外部依赖，或明确说明所需环境。
- 本仓库仅用于归档和展示，不对页面内容的准确性、安全性或适用性作任何保证。
